# Мастер обоев (Telegram Bot)

Готовый проект для деплоя на Render.com. Бот отправляет обои, поддерживает VIP-категорию,
и позволяет администратору загружать новые изображения прямо в чате.

## Быстрый старт (Render)
1. Создай репозиторий и загрузь туда содержимое папки `master_oboev`.
2. На Render создай новый **Worker** (или используй render.yaml).
3. В настройках сервиса Render добавь переменные окружения:
   - `BOT_TOKEN` — токен Telegram-бота
   - `ADMIN_PASS` — пароль администратора
4. Запуск: `python bot.py` (Render использует startCommand из render.yaml).

## Локальный запуск
1. Установи зависимости:
   ```
   pip install -r requirements.txt
   ```
2. Экспортируй переменные окружения:
   - Linux/macOS:
     ```
     export BOT_TOKEN="твой_токен"
     export ADMIN_PASS="пароль_админа"
     ```
   - Windows (PowerShell):
     ```
     $env:BOT_TOKEN="твой_токен"
     $env:ADMIN_PASS="пароль_админа"
     ```
3. Запусти:
   ```
   python bot.py
   ```

## Примечания
- База данных `database.db` создаётся автоматически в корне проекта.
- Оплата VIP реализована вручную (админом) — можно расширить интеграцией Telegram Payments.
